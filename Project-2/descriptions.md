## MovieDB

MovieDB is a full-stack application for managing a movie database, similar to IMDB. The backend is built with Java using Spring Data for database operations and Spring Web for the REST API, while the frontend uses Angular for a responsive user interface. The application follows a 3-tier architecture, separating API endpoints, business logic, and data access layers on the backend, with a component-based frontend that consumes the REST API. Users can perform operations such as searching for movies, retrieving movie details, and managing movie information through an intuitive web interface.
