## MovieDB REST API

MovieDB is a REST API for managing a movie database, similar to IMDB. Built with Java, it uses JDBC for database operations and Javalin as the web framework. The application follows a 3-tier architecture, separating API endpoints, business logic, and data access layers. Users can perform operations such as searching for movies, retrieving movie details, and managing movie information through standard RESTful endpoints.
